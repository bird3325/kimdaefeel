# 김대필 (ContextWrite) 에이전트 개발 규칙 (Project Rules)

본 규칙은 **김대필 (ContextWrite)** 프로젝트를 개발하고 개선하는 과정에서 에이전트가 반드시 준수해야 하는 지침입니다.

---

## 1. 최우선 사용자 규칙 (Global Rules)
- **언어 설정:** 모든 대화, 설명, 코드 주석, 아티팩트는 한국어로 알기 쉽게 구성합니다.
- **레이아웃 보존:** 요청 페이지 또는 기존 소스코드의 UI/UX 레이아웃과 스타일은 변경하지 않고, 기능 개발 및 버그 수정과 같은 로직 영역만 안전하게 수정합니다.
- **디자인 가이드라인 준수:** 신규 UI 개발 또는 기존 UI와의 연결이 필요할 때만 `DESIGN_GUIDELINE.md`에 명시된 규격(컬러 `#1E40AF`, `#0D9488`, 줄 간격 1.25 등)을 참조하여 정확하게 적용합니다.

---

## 2. Manifest V3 & 아키텍처 규칙

### 2.1 파일 구조 분리
- **Content Script:** DOM 요소 및 activeElement 탐색, 인라인 미니 플로팅 아이콘 제어 및 이벤트 전송 전담.
- **Background Worker:** 외부 AI API와의 연동 처리, 판단 로직(LLM 및 정규식 하이브리드 필터), 로컬 캐시 관리 전담.
- **chrome.storage.local:** API Key 및 사용자 프로필, 설정 상태를 브라우저 내에 안전하게 저장.

### 2.2 보안 및 프라이버시
- 비밀번호 입력창(`type="password"`), 신용카드 입력창, 지정된 금융 도메인 등 민감한 폼 데이터를 감지했을 때는 자동으로 텍스트 수집 및 스위칭 대상에서 즉시 제외하는 블랙리스트 필터를 최우선 실행합니다.

---

## 3. 성능 및 코드 안전 제어 (Gaurdrails)
- **디바운스(Debounce):** 키 입력 또는 페이지 전환 감지 시 `500ms` 디바운스를 강제하여 무분별한 백그라운드 메시징 및 API 호출을 차단합니다.
- **사실 관계 안전 잠금 (`{{Fact_Protection_Lock}}`):**
  - Humanize(자연화) 변환 프롬프트 조립 시, 사용자가 제공한 원본 에피소드의 사실 관계 데이터(연도, 사명, 점수, 프로젝트 명 등)가 AI 변환 과정에서 유실되거나 왜곡되지 않도록 하는 가드레일 프롬프트를 명시적으로 삽입합니다.
