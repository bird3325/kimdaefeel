개인정보처리방침

본 개인정보처리방침은 크롬 확장 프로그램 '김대필 (ContextWrite)'(이하 '서비스')의 이용과 관련하여, 서비스가 사용자의 개인정보를 어떻게 처리하고 보호하는지 안내하기 위해 작성되었습니다.

1. 개인정보의 수집 항목 및 목적

서비스는 사용자가 수동으로 선택하여 요청한 작문 및 변환 기능의 수행을 위한 목적에 한해서만 최소한의 정보를 수집 및 처리하며, 이를 마케팅이나 외부 제3자에게 제공할 목적으로 무단 수집하지 않습니다.

  • AI 자연화 변환용 초안 데이터: 사용자가 자연스러운 문장으로 윤문을 요청하기 위해 입력한 텍스트 및 관련 문맥(URL 패턴, 작성 모드 등)
  • AI API 연동 설정 정보: 사용자가 설정 페이지에 직접 등록한 API Key 및 연동 설정(브라우저 로컬 스토리지에 저장되어 외부로 절대 공유되지 않음)

2. 개인정보의 저장 및 처리 방식 (로컬 지향 보안)

서비스는 프라이버시 보호를 최우선으로 하여 설계되었습니다.

  • 브라우저 내 로컬 보존: 사용자의 API Key 및 작성 프로필(어조, 경력, 에피소드 등) 정보는 사용자의 기기 내부 저장소(chrome.storage.local)에 암호화 또는 격리된 상태로 안전하게 보관되며, 서비스 제공자의 별도 서버로 전송되거나 축적되지 않습니다.
  • 안전한 외부 API 전송: 변환 요청이 발생할 시 사용자가 지정한 공식 AI 제공사(OpenAI, Google Gemini, NVIDIA NIM API)의 공인 규격 보안 서버로만 실시간 연동 처리됩니다.

3. 민감 정보 차단 필터 (Privacy Guard)

서비스는 비밀번호 입력 필드(type="password"), 신용카드 폼, 인터넷 뱅킹 및 금융 관련 도메인의 폼 데이터를 자동으로 감지하여 데이터 분석 및 수집 범위에서 즉각 원천 제외하는 블랙리스트 가드레일을 기본 실행합니다.

4. 정보주체의 권리 및 연락처

사용자는 언제든지 크롬 확장 프로그램 삭제 및 설정 초기화를 통해 로컬에 저장된 모든 API 설정 및 수집 데이터를 스스로 완전히 영구 파괴하실 수 있습니다.

서비스와 관련하여 개인정보 관련 문의사항이나 피드백이 있으신 경우, 개발자 지원 창구를 통해 연락해 주시기 바랍니다.
